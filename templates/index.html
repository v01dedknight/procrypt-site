Популярные вопросы

На каких аппаратных платформах доступен ProCrypt?
В данный момент ProCrypt доступен для компьютеров. Вы можете отслеживать создание новых версий на GitHub (подробнее во вкладке "Разработчик")
Как установить ProCrypt?
Всё уже собрано в 1 архив. Установить - на вкладке "Скачать" вверху страницы
Как научиться пользоваться приложением?
Ваше время ценно. В приложении предусмотрены справки, с подробным описанием функционала всего приложения.
Как работает шифрование в ProCrypt?
ProCrypt использует алгоритм шифрования AES-256 для защиты всей базы данных. При шифровании создаётся специальный файл, для усиления защиты. Ваши данные защищены мастер-паролем
Как происходит оценка "защищённости" пароля?
На вкладке "Проверка пароля" ваш пароль проверяется при помощи специальной математической формулы, специально составленной, для имитации перебора методом brute-force (эти данные можно использовать только для справки)
Как я могу помочь проекту?
Всегда приятно получить поддержку в нужную минуту. Обратите внимание на реквизиты Разработчика, указанные на этом сайте.
